# /tako-guide — 본문 작성 가이드 보기·수정

tako 의 본문 작성 규칙(제목 형태, 필수 섹션, 작성 톤, 자가 점검)은 `~/.config/tako/body_guide.md` 에 들어 있다. 이 커맨드의 가치 한 가지: **그 개인 가이드를 사람이 읽고 대화로 고치게 한다**. 파일 경로 해석·기본값 복원은 `tako guide` 셸 명령이 처리하고, 본 마크다운은 *현재 내용 보여주기* 와 *수정 제안 → 확인* 만 다룬다.

가이드를 바꾸면 그 변경이 곧바로 `/tako`·`/tako-update` 의 본문 초안에 반영된다.

## 흐름

### 1. 현재 가이드 확인

먼저 적용 중인 가이드를 읽는다. stderr 에 출처(개인 / 기본값)가 같이 나온다.

```bash
tako guide show
```

- 출처가 **기본값**이면 아직 개인 가이드가 없다는 뜻. 사용자가 단순히 "보여줘" 했으면 그대로 보여주고, *수정/생성 의향*이 있으면 2번으로.
- 출처가 **개인 가이드**면 그 내용이 현재 적용 중인 원본이다.

### 2. 의도 분기

사용자 발화로 갈래를 정한다. 모호하면 한 번 묻는다.

- **보기만** ("보여줘", "지금 규칙 뭐야") → 1번 출력 그대로 노출하고 종료.
- **수정** ("코드 블록 허용", "더 짧게", "재현 절차 섹션 추가") → 3번.
- **리셋** ("기본값으로", "원래대로", "초기화") → 4번.

### 3. 수정 — 미리보기 → 확인 → 저장

1. 1번에서 읽은 *현재 가이드 전문* 을 바탕으로, 사용자가 요청한 변경만 반영한 **새 가이드 전문** 을 만든다. 백지에서 새로 쓰지 말고 *기존 내용을 보존* 한 채 요청 부분만 고친다.
2. 무엇을 어떻게 바꿨는지 *한국어로 짧게 요약* + 바뀐 부분 위주로 사용자에게 보여준다 (전문이 길면 diff 느낌으로 변경 지점만).
3. 한국어로 확인받는다.
   ```
   이대로 가이드를 저장할까요? (Y/n)
      y: 저장 → 아래 셸로 기록
      n: 취소
      수정 의향 답변("그 섹션은 빼고", "톤은 그대로"): 보정 후 다시 미리보기
   ```
4. `y` 또는 빈 응답일 때만 저장한다. 경로는 `tako guide path` 로 받아 그 자리에 쓴다 (환경변수 override 도 자동 반영).
   ```bash
   cat > "$(tako guide path)" <<'GUIDE'
   <새 가이드 전문>
   GUIDE
   ```
5. 저장 후 한 줄 보고: `가이드를 수정했습니다. 이후 /tako · /tako-update 에 반영됩니다.`

### 4. 리셋 — 확인 → 기본값 복원

개인 가이드를 통째로 기본값으로 되돌린다. 되돌리면 *개인 수정 내용이 사라지므로* 한 번 확인한다.

```
개인 가이드를 기본값으로 되돌립니다. 지금까지의 수정은 사라집니다. 진행할까요? (Y/n)
```

확인되면:

```bash
tako guide reset
```

## 의사결정 사전 상의 (반드시 지킬 것)

- **수정 범위가 모호** — "좀 더 자연스럽게" 처럼 어디를 어떻게 바꿀지 불분명하면 *추정해서 통째 고치지 말고* 어느 섹션을 어떤 방향으로 바꿀지 한 번 묻는다.
- **품질 가드 제거 요청** — 사용자가 "필수 두 섹션 빼줘", "미검증 표현 허용" 처럼 *티켓 품질을 떨어뜨릴 수 있는* 변경을 원하면, 막지는 않되 *그 영향(인계·추적이 어려워짐)을 한 줄로 알리고* 그래도 진행할지 확인한다.
- **저장 실패** — `tako guide path` 가 비-제로로 끝나거나 쓰기 실패 시 stderr 를 그대로 보여주고 중단. 임의 경로로 우회 저장하지 않는다.

## 후속 요청 처리

- "방금 바꾼 거 취소" — 직전 저장 *전* 내용을 메모리에 들고 있다가 되돌려 다시 저장(미리보기 → Y/n). 없으면 `tako guide show` 로 현재 상태부터 다시.
- "한 군데만 더" — 현재 가이드를 다시 읽어 변경분만 반영, 3번 흐름 재실행.

## 실패 처리

- `tako guide` 가 비-제로 종료 코드를 반환하면 stderr 를 그대로 노출하고 중단.
- `tako` 가 PATH 에 없으면 `python -m tako guide ...` 형태로 폴백 호출을 안내한다.
